# 作業ログ 2026-03-28

## 概要

Ebitengine を使ったクトゥルフ風スネークゲームのプロジェクトを 0 から立ち上げ、
スプライト画像・BGM・SEを統合したゲームとして動作するところまで実装した。

## 作業内容

### 1. プロジェクト初期構築

- `go.mod` 作成（モジュール名: `snakegame_with_ebitengine`）
- Ebitengine v2 を依存に追加、`go mod tidy` で解決
- `main.go` にスネークゲームの基本ロジックを実装（約180行）
  - `ebiten.Game` インターフェース（`Update`, `Draw`, `Layout`）
  - 蛇の移動、矢印キー操作、食べ物生成、衝突判定、スコア、ゲームオーバー＆リスタート
  - 描画は `ebitenutil.DrawRect()` による色付き矩形

### 2. ドキュメント作成

- `docs/ebitengine-guide.md` — Ebitengine 初心者向けリファレンスガイド
  - Game インターフェース、座標系、描画API、入力処理、最小サンプルコード
- `docs/milestones.md` — 実装マイルストーン（M0〜M6）
- `README.md` — プロジェクト概要、遊び方、セットアップ手順

### 3. 日本語コメント追加

- `main.go` の全セクションに初心者向けの詳細な日本語コメントを付与
- セクション区切り線（`──────`）で構造を明確化
- ロジック変更なし、コメントのみの変更としてコミット

### 4. スプライト画像の作成と統合

- Gemini で生成したスプライトシート画像からゲーム用素材を抽出
  - 詳細は `docs/sprite_extraction_log.md` を参照
- 最終的なアセット一覧（各128x128px）:
  - 頭部: `head_up.png`, `head_down.png`, `head_left.png`, `head_right.png`（方向別4枚）
  - 胴体: `body_vertical.png`, `body_horizontal.png`（直線2枚）
  - カーブ: `body_segment_curved_top_left.png`, `body_segment_curved_top_right.png`, `body_segment_curved_bottom_left.png`, `body_segment_curved_bottom_right.png`（曲がり角4枚）
  - 食べ物: `food_necronomicon.png`, `food_brain.png`, `food_elder_sign.png`（3種類）
  - 背景: `background.png`（640x480px）

### 5. ゲームコードのスプライト対応

- `embed` パッケージでアセットをバイナリに埋め込み
- `DrawRect` → `DrawImage` に変更、128px スプライトを `gridSize` にスケーリングして描画
- 頭部: 方向別スプライトをマップで管理（回転処理不要）
- 体: 前後セグメントの位置関係から直線/カーブを自動判定
- 食べ物: 3種類からランダム選択
- 背景: `background.png` を `ColorScale` で30%に減光して描画

### 6. ゲームバランス調整

- `gridSize`: 20 → 32 に拡大（スプライトの視認性向上、グリッド 32x24 → 20x15）
- `moveInterval`: 8 → 15 に変更（移動速度 約7.5回/秒 → 約4回/秒）
- 背景の明るさを30%に抑制（スプライトとのコントラスト改善）

### 7. 一時停止機能の追加

- Space キーでポーズ/再開をトグル（`inpututil.IsKeyJustPressed` で瞬間判定）
- ポーズ中は半透明オーバーレイ + "PAUSED" メッセージを表示
- ゲームオーバー時のリスタートは Enter キーのみに変更（Space との衝突回避）

### 8. .gitignore 作成

- ビルドバイナリ（`snakegame_with_ebitengine`）と `.DS_Store` を除外

### 9. BGM の追加

- BGMファイル: `assets/bgm.mp3`（Wet Tentacles on Stone）
- `audio` / `audio/mp3` パッケージで MP3 デコード → `NewInfiniteLoop` で無限ループ再生
- 音量30%（`SetVolume(0.3)`）
- ゲーム起動時に自動再生開始

### 10. SE（効果音）の追加

- Pixabay からフリー素材を `yt-dlp` で取得:
  - `se_eat.mp3` — 食べ物を食べた時の効果音（[Pixabay: game-eat-sound-83240](https://pixabay.com/sound-effects/game-eat-sound-83240/)）
  - `se_gameover.mp3` — ゲームオーバー時の効果音（[Pixabay: videogame-death-sound-43894](https://pixabay.com/sound-effects/videogame-death-sound-43894/)）
- 起動時に MP3 → PCM にデコードしてメモリに保持、再生時は `NewPlayer` で即座に鳴らす
- SE音量50%（`SetVolume(0.5)`）

### 11. 尻尾スプライトの追加

- `tail_up.png`, `tail_down.png`, `tail_left.png`, `tail_right.png`（各128x128px）
- 方向別マップ `tailSprites` で管理、`init()` で読み込み
- 尻尾の向き = 前のセグメントの**反対方向**（尻尾の先端が伸びる方向）
- 初回実装で向きが全方向逆だったため修正

### 12. GitHub Pages 公開

- `GOOS=js GOARCH=wasm go build` で WASM ビルド成功を確認（16MB）
- Go ランタイムから `wasm_exec.js` をコピー
- `index.html` 作成（4:3 アスペクト比、ダークテーマ、操作説明付き）
- `.github/workflows/deploy.yml` 作成（main push → WASM ビルド → Pages デプロイ）
- リポジトリを Private → Public に変更（`gh repo edit --visibility public`）
- `gh api` で GitHub Pages 有効化（Source: GitHub Actions）
- デプロイ成功（57秒）: https://webzawa.github.io/snakegame_with_ebitengine/

### 13. スマホ・タブレット対応

- `main.go`: スワイプ操作（`handleTouch()`）とタップリスタート（`isTapped()`）を追加
  - タッチ開始位置から16px以上スワイプで方向確定、連続スワイプ対応
  - 180度反転防止もスワイプに適用
- `index.html`: モバイル向け最適化
  - `viewport`: ピンチズーム無効化
  - `touch-action: none` + `touchmove` の `preventDefault()` でスクロール完全無効化
  - レスポンシブサイズ調整
  - `@media (pointer: coarse)` でモバイル/デスクトップの操作ヒント切替
