# 作業ログ 2026-03-28

## 概要

Ebitengine を使ったクトゥルフ風スネークゲームのプロジェクトを 0 から立ち上げ、
スプライト画像を使ったゲームとして動作するところまで実装した。

## 作業内容

### 1. プロジェクト初期構築

- `go.mod` 作成（モジュール名: `snakegame_with_ebitengine`）
- Ebitengine v2 を依存に追加、`go mod tidy` で解決
- `main.go` にスネークゲームの基本ロジックを実装（約180行）
  - `ebiten.Game` インターフェース（`Update`, `Draw`, `Layout`）
  - 蛇の移動、矢印キー操作、食べ物生成、衝突判定、スコア、ゲームオーバー＆リスタート
  - 描画は `ebitenutil.DrawRect()` による色付き矩形

### 2. ドキュメント作成

- `docs/ebitengine-guide.md` — Ebitengine 初心者向けリファレンスガイド
  - Game インターフェース、座標系、描画API、入力処理、最小サンプルコード
- `docs/milestones.md` — 実装マイルストーン（M0〜M6）
- `README.md` — プロジェクト概要、遊び方、セットアップ手順

### 3. 日本語コメント追加

- `main.go` の全セクションに初心者向けの詳細な日本語コメントを付与
- セクション区切り線（`──────`）で構造を明確化
- ロジック変更なし、コメントのみの変更としてコミット

### 4. スプライト画像の作成と統合

- Gemini で生成したスプライトシート画像からゲーム用素材を抽出
  - 詳細は `docs/sprite_extraction_log.md` を参照
- 最終的なアセット一覧（各128x128px）:
  - 頭部: `head_up.png`, `head_down.png`, `head_left.png`, `head_right.png`（方向別4枚）
  - 胴体: `body_vertical.png`, `body_horizontal.png`（直線2枚）
  - カーブ: `body_segment_curved_top_left.png`, `body_segment_curved_top_right.png`, `body_segment_curved_bottom_left.png`, `body_segment_curved_bottom_right.png`（曲がり角4枚）
  - 食べ物: `food_necronomicon.png`, `food_brain.png`, `food_elder_sign.png`（3種類）
  - 背景: `background.png`（640x480px）

### 5. ゲームコードのスプライト対応

- `embed` パッケージでアセットをバイナリに埋め込み
- `DrawRect` → `DrawImage` に変更、128px スプライトを `gridSize` にスケーリングして描画
- 頭部: 方向別スプライトをマップで管理（回転処理不要）
- 体: 前後セグメントの位置関係から直線/カーブを自動判定
- 食べ物: 3種類からランダム選択
- 背景: `background.png` を `ColorScale` で30%に減光して描画

### 6. ゲームバランス調整

- `gridSize`: 20 → 32 に拡大（スプライトの視認性向上、グリッド 32x24 → 20x15）
- `moveInterval`: 8 → 15 に変更（移動速度 約7.5回/秒 → 約4回/秒）
- 背景の明るさを30%に抑制（スプライトとのコントラスト改善）

### 7. 一時停止機能の追加

- Space キーでポーズ/再開をトグル（`inpututil.IsKeyJustPressed` で瞬間判定）
- ポーズ中は半透明オーバーレイ + "PAUSED" メッセージを表示
- ゲームオーバー時のリスタートは Enter キーのみに変更（Space との衝突回避）
